AERFS README 2025-11-21

anaFunctions.R holds functions to calculate and output AERFS estimates
etmFunctions.R holds functions to transform simulated data to appropriate format for etm() and runs etm() on transformed data
simFunctions.R holds functions to simulate data
masterFunction.R holds a function to run all helper functions in anaFunctions.R, etmFunctions.R, simFunctions.R and outputs simulation results
simSettings.R calls masterFunction and runs the simulation
